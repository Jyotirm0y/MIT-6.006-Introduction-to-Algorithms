## Brute-force method 
```
def max_sub_array(A):
    left=right=0
    max_sum=-999
    array_sum=0
    for i in range(0,len(A)):
        for j in range(i,len(A)):
            array_sum=sum(A[i:j+1])
            if array_sum>max_sum:
                max_sum=array_sum
                left=i
                right=j
    return (i,j,max_sum)  
```
